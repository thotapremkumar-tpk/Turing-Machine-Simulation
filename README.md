# Turing-Machine-Simulation
# 🧠 Universal Turing Machine Simulator

A visual and interactive simulator for understanding **Turing Machines** and **Formal Languages**.

---

## 🚀 Features

- 🎯 Supports multiple languages:
  - Palindrome (w = wᴿ)
  - Equal number of 0s and 1s
  - aⁿbⁿ
  - aⁿbⁿcⁿ
  - Even number of 0s
  - ww (string duplication)
  - Prime length strings
  - Arithmetic expression validation

- 🖥️ Interactive UI:
  - Step-by-step execution
  - Auto run with speed control
  - Tape visualization
  - State diagram visualization
  - Transition table display
  - Execution log

- 🎨 Clean and professional dark UI
