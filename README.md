# Turing Machine Simulation

## Description

This project is a simple web-based simulation of a Turing Machine. It demonstrates how different formal languages are processed step by step using basic computational logic. The simulator is designed to make abstract concepts easier to understand by visualizing how input strings are evaluated.

It currently supports languages such as palindrome checking and equal number of 0s and 1s, helping users explore the working of theoretical computation models in an interactive way.

---

## How to Run

1. Download or clone this repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Open the project folder
3. Open `index.html` in any web browser

No additional setup or installation is required.

## Features

* Simulates Turing Machine behavior step by step
* Supports multiple language checks such as palindrome and equal 0s and 1s
* Simple and interactive user interface
* Provides instant output and visualization
* Easy to understand for beginners
